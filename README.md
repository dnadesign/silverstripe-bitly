# silverstripe-bitly
Provides [Bitly](https://bitly.com) url generation for all pages.

## Requirements
* silverstripe/cms: ^4

## Installation
```bash
$ composer require vulcandigital/silverstripe-bitly
```

Afterwards, run `dev/build`
